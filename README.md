# Grand Tour

A quick and dirty collaborative test of Strava's API. Do not use.


## Getting Started

Create the database:

```bash
cat schema.sql | sqlite3 app.db
```

You can find your client id and secret on the [my API application](https://www.strava.com/settings/api)
page.

```toml
# config.toml
id = 123
secret = "baaaaaad"
```
